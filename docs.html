Class FunctionLib

Public Function GetMileageFromLocations(fromcode As String, tocode As String) As Decimal


Convert two locations into the distance between them. Returns the mileage as a decimal.


Public Function GetTimeFromLocations(fromcode As String, tocode As String) As String


Convert two locations into the time it takes to travel between them. Returns the time as a string without spaces.


Public Function GetJsonValue(jsonText As String, name As String, Optional skipCount As Integer = 0) As String


Get the value of a JSON entry. Returns the value as a string.


Public Function RemoveStringLetters(original As String) As String


Removes letters from a string. Returns the modified string.


Public Function RemoveStringNumbers(original As String) As String


Removes numbers from a string. Returns the modified string.


Public Function RemoveStringSymbols(original As String) As String


Removes symbols from a string. Returns the modified string.


Public Function RemoveIllegalCharacters(dirty As String) As String


Remove characters not recognized by filters. Returns clean string.


Public Function FormatUrlMileage(fromAddr As String, toAddr As String, apiKey As String) As String


Format url for mileage from postal codes. Returns the formated URL as a string.


Public Function GetRawJSON(url) As String


Retrieve a raw JSON string from a URL. Returns the JSON as a string. WARNING: JSON may contain dirty characters. Call RemoveIllegalCharacters() to clean.


Public Function Init() As Boolean


Initialize the API interface. Returns true if run successfully.
